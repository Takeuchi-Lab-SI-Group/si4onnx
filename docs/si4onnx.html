si4onnx.data module

class si4onnx.data.SyntheticDataset(n_samples, shape, loc=0, scale=1, local_signal=0, local_size=None, seed=0)

Bases: Dataset

A PyTorch Dataset for generating synthetic data with iid Gaussian noise and local signals.

This dataset generates samples containing Gaussian noise with optional local signals added at random positions.

The dataset generates three components for each sample:

  1. Data: Tensor containing Gaussian noise with optional local signals

  2. Mask: Binary tensor indicating the position of local signals

  3. Label: Binary value indicating presence of local signal

data

Array containing the generated synthetic data.

Type:

numpy.ndarray

masks

Array containing the masks for local signals.

Type:

numpy.ndarray

labels

Array containing the binary labels.

Type:

numpy.ndarray

Examples

>>> dataset = SyntheticDataset(
...     n_samples=1000,
...     shape=(32, 32),
...     loc=0,
...     scale=1,
...     local_signal=2.0
... )
>>> data, mask, label = dataset[0]
>>> print(data.shape, mask.shape, label.shape)
torch.Size([32, 32]) torch.Size([32, 32]) torch.Size([])
si4onnx.data.generate_iid_data(n_samples, shape, loc=0, scale=1, local_signal=0, local_size=None, seed=0)

Generate synthetic data with iid Gaussian noise and local signals.

Parameters:

  • n_samples (int) – The number of samples.

  • shape (tuple) – The shape of the data (channels, height, width).

  • loc (float, optional) – The mean of the Gaussian noise. Default is 0.0.

  • scale (float, optional) – The standard deviation of the Gaussian noise. Default is 1.0.

  • local_signal (float, optional) – The strength of the local signal to add. Default is 0.0.

  • local_size (int or None, optional) – The size of the local signal region. If None, uses min(height, width) // 3.

  • seed (int, optional) – The seed for reproducibility. Default is 0.

Returns:

  • np.ndarray – The generated data of shape (size, channels, height, width).

  • np.ndarray – Binary masks indicating signal locations of shape (size, channels, height, width).

  • np.ndarray – Binary labels indicating presence of signal of shape (size,).
